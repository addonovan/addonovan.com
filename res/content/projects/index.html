addonovan.com

Github Languages Rust HTML & CSS Technologies Actix-Web Handlebars

arm7.js

Github Language JavaScript

A (limited) Arm7 interpreter written in JavaScript. This can simulate some basic instructions and display how the 32-bit registers act during the program execution, all while showing the current line being executed.

The Arm7 assembly is parsed before it is interpreted, by using a parser generated from a PEG.js grammar.

FAT32 Image Reader

Github Language C

This was a school assignment for my undergraduate Operating Systems class in Spring 2018. The program will mount a FAT32 file system image and drops you into an interactive command-line shell which will let you interact with the file system (reads only).

The assignment comes with an implementation of a custom memory allocator which tracks when blocks are allocated and freed when in debug mode.

To make sure each part of the program was working correctly, I wrote a unit testing system along with the program and practiced test-driven development for this project. The unit testing framework is pretty nice to use, and will run all unit tests under a separate process with a SIGSEGV handler attached to catch segfaults. Shared memory is also employed to keep track of the total number of assertions (and failures) throughout the program.

Github Language C++11 Technologies FLTK OpenSplice DDS

This was a school assignment for my undergraduate Software Engineering course in Spring 2018. We were tasked with specifying, writing, and testing a networked game of Blackjack which uses the FLTK library for the GUI and OpenSplice DDS for networked communication, both prescribed by the professor.

gvengine

Github Language Java Technologies LWJGL 2 OpenGL

gvengine is a portmanteau of its former name, GVEN, and engine. As its original acronym implies, it is a generic voxel engine, with very simplistic rendering capabilities. This project was created using OpenGL from scratch (via LWJGL) and displays cubes of different colors organized into 16-block cubes called chunks, which are themselves organized into 16-chunk cubes called regions.

None of this project is particularly well designed or efficient, as I didn't really understand matrices at the time or any of the math surrounding them. I started the project when I was a sophomore in high school at age 16. This was also my first real experience using git (or any vcs for that matter), so there are multiple commits (including the latest one as of writing this, 28162c1 , which do not run properly. The project also doesn't have a build system, so that's really fun to set up every time 🙄.

This project include an asynchronous task scheduler which makes heavy use of reflections, because jdk 1.7.x didn't have lambda functions yet, and I thought making Runnables was ugly (which they are, but it's pretty bad to use reflections instead).

Mav Shell

Github Language C

This was a school assignment for my undergraduate Operatin Systems class in Spring 2018. The program is a simple UNIX-style shell, like sh. The program comes with following built-in commands:

  • cd
  • pwd prints working directory
  • history shows previously run commands
  • showpids shows pids of previously run commands
  • !n runs the nth previous command

robosim

Github Languages Java Python Technologies Box2d Jython

Robosim was designed to be a top-down simulation of a robot for FTC tournaments. Intended to be used to test programs which would control the robot by reading from its sensors and powering motors, the robot's script was written in Python and the scene would be rendered using LibGDX.

Although the program didn't end up fully working out (Box2d was apparently not meant to simulate top-down physics, which I found out during the course of this project), the program was fun to write and I learned how to integrate a friendly Python interface into a program.
