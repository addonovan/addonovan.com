kftc is a complete revamp of the standard FTC library in a way that makes it much more kotlin friendly, while also adding many useful utilities and fixing many annoyances and issues with the standard library.

Features

Kotlin features many useful libraries and utilities, such as:

  • A revamped OpMode lifecycle
  • A build-in configuration framework
  • An asynchronous task management system
  • Better hardware access and extension systems
